# Spiki Tamagotchi

**귀여운 스피키와 함께 코딩하세요!**

VSCode에서 키우는 귀여운 가상 펫 스피키입니다. 코딩하면 스피키가 행복해지고, 레벨업하면 친구들이 늘어나요!

![Spiki Preview](media/images/spiki1.png)

## Features

### Virtual Pet System
- **Stats**: 행복도, 포만감, 에너지 관리
- **Level Up**: 코딩할수록 경험치 획득, 레벨업!
- **Multiply**: 레벨업하면 새 친구가 생겨요

### Interactions
- **Feed**: 밥주기로 포만감 UP
- **Play**: 놀아주기로 행복도 UP
- **Pet**: 쓰다듬기로 애정 표현
- **Sleep**: 재우면 에너지 회복
- **Drag**: 잡아당기면 늘어나요! "머리 잡아 당기지 마세요~"

### Coding Detection
- 타이핑하면 스피키가 응원해요
- 파일 저장하면 보상
- 디버그 시작하면 반응
- 터미널 열면 놀라요

### Editor Integration
- 에디터 gutter에 스피키 등장
- 코드 사이를 돌아다녀요
- 여러 마리가 동시에!

### Sound Effects
- 10가지 귀여운 음성
- "좋아요!", "스피키~", "에?" 등

## Commands

| Command | Description |
|---------|-------------|
| `Spiki: Show Spiki` | 스피키 패널 열기 |
| `Spiki: Feed Spiki` | 밥주기 |
| `Spiki: Play with Spiki` | 놀아주기 |
| `Spiki: Pet Spiki` | 쓰다듬기 |
| `Spiki: Toggle Editor Spiki` | 에디터 스피키 켜기/끄기 |
| `Spiki: Add Spiki to Editor` | 에디터에 스피키 추가 |

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `spiki.showStatusBar` | `true` | 상태바에 스피키 표시 |
| `spiki.soundEnabled` | `true` | 소리 효과 |
| `spiki.autoFeed` | `true` | 코딩 활동 자동 보상 |
| `spiki.editorSpiki` | `true` | 에디터에 스피키 표시 |

## Tips

- 스피키를 잡고 위로 끌어당겨보세요!
- 행복도가 낮으면 친구들이 떠나요...
- 레벨업할 때마다 새 친구가 와요
- 코딩을 많이 하면 스피키가 좋아해요
